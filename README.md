# NetTrace — Graph-Based Fraud Detection

NetTrace is a graph-based fraud detection system that models financial transactions as a network rather than analyzing them individually. Accounts are represented as nodes and transactions as directed edges, allowing the system to capture relationships between entities. Using graph-based machine learning (GraphSAGE), anomaly detection (Isolation Forest), and ensemble scoring, the system identifies fraudulent accounts based on both their behavior and their position within the transaction network.
